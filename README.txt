Download the .apk file in your Android device and install it to run.
***********************************************************************
If you want to modify the code:
  
1) Find the relevant source files(.java) in  gtucalc/src/com/example/gtucalc/
2) Find the relevant .xml files in gtucalc/res/layout
3) Find the image files in gtucalc/res/drawable
4) Find the Android Manifest file in gtucalc/
5) Add all the mentioned files in your project once you create one called     "gtucalc".
6) Hopefully you will find this helpful!!
